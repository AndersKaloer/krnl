41 NB NB ABOUT TIMERS
42 
43 Pins 5 and 6: controlled by timer0
44 Pins 9 and 10: controlled by timer1
45 Pins 11 and 3: controlled by timer2
46 
47 On the Arduino Mega we have 6 timers and 15 PWM outputs:
48 
49 Pins 4 and 13: controlled by timer0
50 Pins 11 and 12: controlled by timer1
51 Pins 9 and10: controlled by timer2
52 Pin 2, 3 and 5: controlled by timer 3
53 Pin 6, 7 and 8: controlled by timer 4
54 Pin 46, 45 and 44:: controlled by timer 5
55 
56 ... from http://arduino-info.wikispaces.com/Timers-Arduino
57 
58 Servo Library uses Timer1.
59  You can’t use PWM on Pin 9, 10 when you use the Servo Library on an Arduino.
60  For Arduino Mega it is a bit more difficult. The timer needed depends on the number of servos.
61  Each timer can handle 12 servos.
62  For the first 12 servos timer 5 will be used (losing PWM on Pin 44,45,46).
63  For 24 Servos timer 5 and 1 will be used (losing PWM on Pin 11,12,44,45,46)..
64  For 36 servos timer 5, 1 and 3 will be used (losing PWM on Pin 2,3,5,11,12,44,45,46)..
65  For 48 servos all 16bit timers 5,1,3 and 4 will be used (losing all PWM pins).
66 
67 Pin 11 has shared functionality PWM and MOSI.
68  MOSI is needed for the SPI interface, You can’t use PWM on Pin 11 and the SPI interface at the same time on Arduino.
69  On the Arduino Mega the SPI pins are on different pins.
70 
71 tone() function uses at least timer2.
72  You can’t use PWM on Pin 3,11 when you use the tone() function an Arduino and Pin 9,10 on Arduino Mega.

210 /**
211  * MACROS MACROS
212  *
213  * PUSHREGS AND POPREGS
214  * is actual staklayout plus task address at top
215  * A push/pop takes 2 cycles
216  * a call takes 3 cycles
217  * ret/iret 3-4 cycles
218  * So a PUSHREGS is 33 instructions(@ 2 cycles) = 66 cycles ~= 66 cycles /"16 MHz" ~= 4.1 usec
219  * So an empty iSR which preserves all registers takes 2*4.1usec + 8-10 cycles (intr + iret) ~= 9 usec
220  * So max isr speed with all regs saved is around 100 kHz but then with no code inside ISR !
221  *
222  * WARNING
223  * The 2560 series has 3 bytes PC the rest only 2 bytes PC !!! (PC. program counter)
224  * and no tall has rampz and eind register
225  *
226  *
227  * REGISTER NAMING AND INTERNAL ADRESSING
228  *
229  * Register addresses
230  * 0x3f SREG
231  * 0x3e SPH
232  * 0x3d SPL
233  * 0x3c EIND
234  * 0x3b RAMPZ
235  * ...
236  * 0x1f R31
237  * etc
238  * 0x01 R1
239  * 0x00 R0
240  *
241  * PC is NOT available
242  */
