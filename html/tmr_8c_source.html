164  /* FOR 16 bits !
165  prescaler in cs2 cs1 cs0
166  0 0 0 none
167  0 0 1 /1 == none
168  0 1 0 /8
169  0 1 1 /64
170  1 0 0 /256
171  1 0 1 /1024
172  16MHz Arduino -> 16000000/1024 = 15.625 ticks/second
173  16MHz Arduino -> 16000000/256 = 62.500 ticks/second
174  -------------------------/64 = 250.000 ticks/second !
175  -------------------------/8 = 2.000.000 ticks/second !
176  -------------------------/1 = 16.000.000 ticks/second !
177 
178 
179  NB 16 bit counter so values >= 65535 is not working
180  so the + value below shall be less than 65535
181 
182  01msec timer /64 + 250 1000 Hz
183  10msec timer /64 + 2500 100 Hz
184  100msec timer /64 + 25000 10 Hz
185  100 usec timer /64 + 250 10 kHz
186  10 usec timer /8 + 20 100 kHz
187  1 usec timer /8 + 2 1 MHz
188 
189  NBNB for 8 bit timers the count down value shall be less than 255
190 
191  EXAMPLE: 100 kHz TCCRxB = 0x02 TCNTx = 20
192  I you want a square wave at 100 KHz sthen isr must run 200 kHz
193 
194  */
